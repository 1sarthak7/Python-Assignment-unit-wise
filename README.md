## Units
 
### Unit 1 — Python Basics
 
[![Unit 1](https://img.shields.io/badge/Browse_Unit_1-blue?style=for-the-badge)](https://github.com/1sarthak7/Python-Assignment-unit-wise/tree/main/Unit-1)
 
Covers the foundational building blocks of Python programming.
 
- Variables, data types, and type casting
- Input/output operations
- Basic arithmetic and logical operators
- Conditional statements (`if`, `elif`, `else`)
 
---
 
### Unit 2 — Control Flow and Loops
 
[![Unit 2](https://img.shields.io/badge/Browse_Unit_2-blue?style=for-the-badge)](https://github.com/1sarthak7/Python-Assignment-unit-wise/tree/main/Unit-2)
 
Focuses on controlling program execution using loops and branching.
 
- `for` and `while` loops
- Loop control: `break`, `continue`, `pass`
- Nested loops and pattern programs
- Iteration techniques
 
---
 
### Unit 3 — Functions and Modules
 
[![Unit 3](https://img.shields.io/badge/Browse_Unit_3-blue?style=for-the-badge)](https://github.com/1sarthak7/Python-Assignment-unit-wise/tree/main/Unit-3)
 
Explores reusability through functions and modular programming.
 
- Defining and calling functions
- Arguments: positional, keyword, default, `*args`, `**kwargs`
- Lambda functions
- Importing and creating modules
 
---
 
### Unit 4 — Data Structures
 
[![Unit 4](https://img.shields.io/badge/Browse_Unit_4-blue?style=for-the-badge)](https://github.com/1sarthak7/Python-Assignment-unit-wise/tree/main/Unit-4)
 
Deep dive into Python's built-in data structures.
 
- Lists, tuples, and their methods
- Dictionaries and sets
- List comprehensions
- Nested data structures
 
---
 
### Unit 5 — Object-Oriented Programming
 
[![Unit 5](https://img.shields.io/badge/Browse_Unit_5-blue?style=for-the-badge)](https://github.com/1sarthak7/Python-Assignment-unit-wise/tree/main/Unit-5)
 
Introduces the OOP paradigm in Python.
 
- Classes and objects
- Constructors (`__init__`) and destructors
- Inheritance, polymorphism, and encapsulation
- Dunder/magic methods
 
---
 
### Unit 6 — File Handling and Exceptions
 
[![Unit 6](https://img.shields.io/badge/Browse_Unit_6-blue?style=for-the-badge)](https://github.com/1sarthak7/Python-Assignment-unit-wise/tree/main/Unit-6)
 
Covers reading/writing files and robust error handling.
 
- Reading and writing text and binary files
- `with` statement and context managers
- Exception handling: `try`, `except`, `finally`, `raise`
- Custom exception classes

## Getting Started
 
### Clone the Repository
 
```bash
git clone https://github.com/1sarthak7/Python-Assignment-unit-wise.git
cd Python-Assignment-unit-wise
```
 
### Run Any Assignment
 
```bash
python Unit-1/assignment1.py
```

## Prerequisites
 
[![Python 3.x](https://img.shields.io/badge/Requires-Python_3.x-3776AB?style=flat-square&logo=python&logoColor=white)](https://www.python.org/downloads/)
 
- Python 3.x installed on your machine
- A code editor such as VS Code, PyCharm, or any terminal-based editor
- No external libraries are required unless specified within a unit's folder

## How to Use
 
1. Navigate to the unit folder that corresponds to your topic of interest.
2. Open the `.py` file for the specific assignment.
3. Read the problem statement in the comments at the top of the file.
4. Run the file using `python filename.py` from your terminal.
5. Modify and experiment with the code to strengthen your understanding.
